    - 每条记录的 attributes 至少包含：
      `certificate_number`, `owner_name`, `building_area`, `usage`, `total_price`
      （可根据需要扩展其它字段，如坐落、单价、所在层等）。
    - 
    Args:
        md_text (str): 已转换为 Markdown 的报告文本（通常为表格片段或包含表格的正文）。

    Returns:
        List[Dict[str, Any]]: 抽取结果列表。每个元素为一条估价对象记录的 attributes 字典。
        若 md_text 为空或未抽取到有效数据，返回空列表 []。

    Notes:
        - 本方法内部使用 `quick_extract` 调用 LangExtract，并依赖 CONFIG 中配置的本地模型参数；
